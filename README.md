## Ferramentas

- VSCode
- NPM
- Angular CLI
- Json-server

## Rodar projeto

No terminal:
 1) Dentro da pasta product-store executar `npm i`
 2) Executar o backend pelo json server com comando `npx json-server db.json`
 3) Executar `npm start`
 4) Abrir o navegador na na URL: `http://localhost:4200/`
